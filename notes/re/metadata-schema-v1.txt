Metadata Schema V1

This is the initial schema sketch for swarm input. It is intentionally simple and append-only. JSONL is the raw transport format; SQLite can be generated from JSONL for joins.

Common row envelope
- schema: string.
- schema_version: 1.
- artifact: string.
- artifact_sha256: string or null.
- extractor: string.
- extractor_version: string or null.
- command: string or null.
- collected_at_utc: string.
- confidence: number.
- data: object.
