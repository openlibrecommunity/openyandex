Swarm Input Plan

Purpose
Prepare deterministic input for agents. Agents should consume metadata and artifacts, not chat context. This keeps reverse engineering reproducible and lets multiple specialists work independently.

Directories
- artifacts/binaries/: installers, unpacked product payloads, PE files, symbols, debug packages. Ignored by git.
- .workflows/metadata/: generated JSONL/SQLite metadata corpus. Ignored by git.
- .workflows/logs/: extractor logs. Ignored by git.
- .workflows/tmp/: temporary extraction output. Ignored by git.
- notes/re/: durable facts, decisions, and evidence summaries. Tracked by git.
- chromium/: local Chromium source snapshot and optional external tooling. Ignored by git.

First corpus files
- .workflows/metadata/baseline.jsonl
- .workflows/metadata/yandex-files.jsonl
- .workflows/metadata/yandex-pe.jsonl
- .workflows/metadata/yandex-strings.jsonl
- .workflows/metadata/chromium-files.jsonl
- .workflows/metadata/chromium-names.jsonl
- .workflows/metadata/github-search-hits.jsonl
- .workflows/metadata/corpus.sqlite

Fact envelope
Every generated JSONL row should include these fields when possible:
- schema: short row type, for example product_version or installed_file.
- schema_version: integer, start at 1.
- artifact: source file or URL the fact came from.
- artifact_sha256: hash of the source artifact when available.
- extractor: tool or script name.
- extractor_version: version string or command output when available.
- command: exact command used, if safe to store.
- collected_at_utc: ISO-8601 timestamp.
- confidence: 0.0 to 1.0.
- data: object with row-specific fields.

Initial agent handoff contract
- Acquisition agents may add artifacts under artifacts/ and facts under .workflows/metadata/.
- Metadata agents must not overwrite JSONL; append or create a new dated shard.
- Reverse agents must cite exact artifact hashes and function/string/resource IDs.
- Naming agents must attach confidence and evidence for every recovered name.
- Patch agents must emit candidate patches separately from confirmed patches.
- Validator agents must report commands, environment, pass/fail, and mismatch evidence.

Do not do yet
- Do not run gclient sync or fetch chromium; it downloads too much.
- Do not import every binary into Ghidra before file inventory identifies priorities.
- Do not treat Yandex LastChange as a Chromium commit unless independent evidence appears.
